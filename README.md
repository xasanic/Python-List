# 🐍 Python List Topshiriqlari

Bu repo Python dasturlash tilida **List** mavzusi bo'yicha 28 ta topshiriqni o'z ichiga oladi.

## 📁 Fayllar tuzilmasi

| Fayl | Mavzu | Tasklar |
|------|-------|---------|
| `task_01_05_list_yaratish.py` | 🟢 List yaratish | Task 01–05 |
| `task_06_10_list_indexing.py` | 🟡 List indexing | Task 06–10 |
| `task_11_15_list_slicing.py` | 🔵 List slicing | Task 11–15 |
| `task_16_18_list_len.py` | 🟠 List len() | Task 16–18 |
| `task_19_28_list_metodlari.py` | 🔴 List metodlari | Task 19–28 |

## 🟢 List Yaratish (Task 01–05)
- **Task 01**: 3 ta meva nomidan iborat oddiy list
- **Task 02**: 5 ta sonli ro'yxat
- **Task 03**: Ichma-ich list (ismlar + yoshlar)
- **Task 04**: `input()` orqali 3 ta ism olish
- **Task 05**: 2x2 ichma-ich ro'yxat

## 🟡 List Indexing (Task 06–10)
- **Task 06**: 3-chi elementni chop etish
- **Task 07**: Oxirgi elementni manfiy indeks bilan
- **Task 08**: 2-chi elementni yangilash
- **Task 09**: Ichma-ich listdagi elementni o'zgartirish
- **Task 10**: Foydalanuvchi kiritgan indeks bo'yicha yangilash

## 🔵 List Slicing (Task 11–15)
- **Task 11**: 3-dan 6-gacha bo'lgan qism
- **Task 12**: Juft indeksdagi elementlar `[::2]`
- **Task 13**: Teskari qilish `[::-1]`
- **Task 14**: Oxirgi 3 ta element `[-3:]`
- **Task 15**: Foydalanuvchidan start/end indeks

## 🟠 List len() (Task 16–18)
- **Task 16**: Uzunlik 5 dan katta bo'lsa "Ko'p element"
- **Task 17**: `append` bilan ism qo'shib, `len()` chiqarish
- **Task 18**: Uzunlik toq → True, juft → False

## 🔴 List Metodlari (Task 19–28)
- **Task 19**: `append()` – element qo'shish
- **Task 20**: `insert()` – boshiga qo'shish
- **Task 21**: `remove()` – element o'chirish
- **Task 22**: `pop()` – oxirgi elementni olish
- **Task 23**: `index()` – indeksini topish
- **Task 24**: `count()` – necha marta qatnashgan
- **Task 25**: `sort()` – saralash
- **Task 26**: `reverse()` – teskari aylantirish
- **Task 27**: `clear()` – tozalash
- **Task 28**: `copy()` – nusxa olish
